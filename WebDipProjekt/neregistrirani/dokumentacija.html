Opis projektnog zadatka

Izrada web aplikacije za dodavanje recepata i uputa za recepte te organiziranje tečajeva za korisnike sa ocjenom rada

Opis projektnog rješenja

Funkcionalnosti koje nisu realizirane:
-Provjera korisničkog imena (AJAX)
-CAPTCHA
-.htaccess ispis korisnika
-Straničenje podataka
-Pretraživanje i sortiranje
-AJAX (HTML, JSON)
-Pomak vremena
-Prilagođavanje mediju i dizajn
-Poboljšanje korisničkog iskustva
-Konfiguracija sustava
-Statistika rada sustava
-Grafički prikazi
-Posebne osobine

Opis skripti, mapa mjesta i navigacijski dijagram

Skripte:
validacija.js - skripta za provjeru unesenih podatka tijekom registracije

CSS:
nav.css - CSS datoteka stranica

Mape:
U korijenskom drijektoriju se nalazi CSS datoteka
U mapi "administrator" se nalaze sve PHP skripte vezane uz uloga administratora
U mapi "klase" se nalaze sve skripte klasa (baza, sesija i dnevnik)
U mapi "korisnik" se nalaze sve PHP skripte vezane uz ulogu korisnika
U mapi "moderator" se nalaze sve PHP skripte vezane uz ulogu moderatora
U mapi "neregistrirani" se nalaze sve PHP skripte vezane uz ulogu neregistriranog korisnika
U mapi "slike" se nalaze sve slike uploadane od korisnika i slike potrebne za stranicu
